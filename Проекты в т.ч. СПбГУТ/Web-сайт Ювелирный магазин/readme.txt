Файл состоит из:
1. Папка с фотографиями
2. Папка с шрифтами Roboto
3. несколько файлов html
4. несколько файлов css
5. несколько файлов js

Обратите внимание:
1. файл "style_main.css" позволяет подключить шрифт Roboto.
"style_main.css" подключен к каждому html документу для оформления шапки страницы и подключение шрифтов одной строчки.

2. каждый html документ может иметь подключенный css документ в соответствии с названием файла. Например, new.html<-style_new.css<-js_new.js.

3. для удобства восприятия, название html документ состоит из имени. название css документ состоит из "style_" и имя. javascript файл состоит из "js_" и имя подключаемого файла

3. обратите внимание на то, что файл "js_registration" подключен к разделу "авторизации"(avtorisation.html) и "регистрации"(registration.html) в связи с тем, что в нем содержатся небольшие и похожие скрипты для разной проверки.

4. Большая часть стиле объектов(разметки) оформлены с помощью классов и размещены в отдельный css документах для удобства редактирования и изменения. Рекомендуется делать для понимания и для удобства редактировать коды.

Главная страница - new.html
